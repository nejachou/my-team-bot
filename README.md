# Rule Based Bot, my team bot

Fork from LINE official site

## How to use

### Install deps

```shell
$ npm install
```

### Configuration

```shell
$ export CHANNEL_SECRET=YOUR_CHANNEL_SECRET
$ export CHANNEL_ACCESS_TOKEN=YOUR_CHANNEL_ACCESS_TOKEN
$ export PORT=1234
```

### Run

```shell
$ node .
```

## Webhook URL

```
https://your.base.url/callback
```
